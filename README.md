# challenge-aido_LF_baseline_behavior_cloning

## Folder Structure

There are 5 main folders: Challenger, Logs, Models, School and Trainer. 

The **duckieChallenger/** is responsible for holding the submission content. You should submit your submission via this folder.

The **duckieLog/** is responsible for keeping all the logs that might be needed for training.

The **duckieModels/** is responsible for holding all previous models you generated. If you have limited disk space, it will be a good idea to clean this folder preiodically.

The **duckieSchool/** is responsible for holding the education script for human to drive the duckiebot.

The **duickieTrainer/** is responsible for hodling the Kera based tensorflow code for triaing the nerual network model needed for Cahllneger.
